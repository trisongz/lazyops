{{ source.app_name }} API returns machine readable error codes, human readable error messages and a link to the docs for more information.

Here is how an error response looks like:

```json
{
  "error": {
    "code": "not_found",
    "message": "The requested resource was not found.",
    "doc_url": "{{ source.app_endpoint }}/docs/api-reference/errors#not-found"
  }
}
```
